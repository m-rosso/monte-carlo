Monte Carlo experiments¶

Probability convergence of IV estimates¶

This project applies Monte Carlo experiments for assessing theoretical results. Here, as an extension of notebook "1 Probability convergence of OLS estimates", the consistency of Instrumental Variables (IV) estimates is illustrated by making use of Monte Carlo methods that generate simulated data generating processes.

The first assumption (IV1) of linearity in parameters is still taken here, leading to a populational linear regression model:

\begin{equation} \displaystyle y = x\beta + \epsilon.1 = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_px_p + \epsilon \end{equation}
Where $x$ is 1 x p, $\beta$ is p x 1, and $\epsilon$ is an assumed additive random error term. Note that this equation is valid for any observation in the population of interest. Given a random sample of $N$ instances: \begin{equation} \displaystyle y = X\beta + \epsilon \end{equation} Where, now, $y$ is N x 1, X is N x p, and $\epsilon$ is N x 1.

The IV estimation is useful when assumption OLS3 ($E(x^T\epsilon) = 0$) is not valid, meaning that some of the input variables are correlated with the error term. As shown in notebook for probability convergence of OLS estimates, this implies that $\beta$ can not be identified. Additionally, $\hat{\beta}$ estimated through OLS will not converge to true $\beta$.

Given a vector of exogenous variables $z$ (1 x p), i.e., variables that do not correlate with the random error term, IV estimation proceeds as follows: \begin{equation} \displaystyle y = x\beta + \epsilon \end{equation} \begin{equation} \displaystyle z^Ty = z^Tx\beta + z^T\epsilon \end{equation} \begin{equation} \displaystyle E(z^Ty) = E(z^Tx)\beta + E(z^T\epsilon) \end{equation} \begin{equation} \displaystyle E(z^Tx)^{-1}E(z^Ty) = \beta + E(z^Tx)^{-1}E(z^T\epsilon) \end{equation}

Therefore, two crucial assumptions (IV2 and IV3), similar to OLS2 and OLS3, are needed for IV to be an appropriate estimation method for the assumed equation:

\begin{equation} \displaystyle rank(E(z^Tx)^{-1}) = p + 1 \end{equation}
\begin{equation} \displaystyle E(z^T\epsilon) = 0 \end{equation}

So: \begin{equation} \displaystyle \beta = E(z^Tx)^{-1}E(z^Ty) \end{equation}

This equation identifies $\beta$, allowing its estimation by replacing expected values by averages:

\begin{equation} \displaystyle \hat{\beta} = \Big((1/N)\sum_{i=1}^Nz_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nz_i^Ty_i = (Z^TX)^{-1}Z^Ty \end{equation}

Note that the number of instruments assumed here is equal to the number of endogenous variables. A more general approach would assume a number larger or equal to the number of endogenous variables, leading to the Two-Stage Least Squares (2SLS) estimator.

Assumptions IV2 and IV3, more than technical constraints, have meaningful interpretation. $rank(E(z^Tx)^{-1}) = p + 1$ implies that the endogenous variables are correlated with the instruments. At the same time, $E(z^T\epsilon) = 0$ implies that these instruments that correlates with relevant variables in $x$ are exogenous. Consequently, instruments $z$ can be seen as capturing the variability in $x$ that does not correlate with $\epsilon$.

Assuming IV1-IV3 not only allows the identification of $\beta$, but also produces an estimate $\hat{\beta}$ which converges in probability to $\beta$, i.e., $\hat{\beta}$ is consistent for estimating $\beta$.

\begin{equation} \displaystyle \hat{\beta} = \Big((1/N)\sum_{i=1}^Nz_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nz_i^Ty_i \end{equation} Using IV1: \begin{equation} \displaystyle \hat{\beta} = \Big((1/N)\sum_{i=1}^Nz_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nz_i^T(x_i\beta + \epsilon_i) \end{equation} Organizing this expression leads to: \begin{equation} \displaystyle \hat{\beta} = \Big((1/N)\sum_{i=1}^Nz_i^Tx_i\Big)^{-1}(1/N)\Big(\sum_{i=1}^Nz_i^Tx_i\beta\Big) + \Big((1/N)\sum_{i=1}^Nz_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nz_i^T\epsilon_i \end{equation} \begin{equation} \displaystyle \hat{\beta} = \beta + \Big((1/N)\sum_{i=1}^Nz_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nz_i^T\epsilon_i \end{equation}
Letting $N \rightarrow \infty$: \begin{equation} \displaystyle \lim_{N \rightarrow \infty} \hat{\beta} = \beta + \lim_{N \rightarrow \infty}\Big((1/N)\sum_{i=1}^Nz_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nz_i^T\epsilon_i \end{equation}

Given the delta method and the law of large numbers, and assuming IV2:

\begin{equation} \displaystyle \lim_{N \rightarrow \infty} \hat{\beta} = \beta + E(z^Tx)^{-1}E(z^T\epsilon) \end{equation} Finally, IV3 implies that: \begin{equation} \displaystyle \lim_{N \rightarrow \infty} \hat{\beta} = \beta \end{equation}

This proof demonstrates how IV produces consistent estimates for $\beta$, following the inverse sequence of steps of the method of moments. However, both approaches assume the same hypotheses and, then, hold the same theoretical results.

Here, we apply general Monte Carlo simulation procedures to illustrate this property.

  1. Modeling the process that should be simulated: the process is depicted by the function $\hat{\beta} = (Z^TX)^{-1}Z^Ty$, which is a function of $X$, $Z$, $\epsilon$ and $\beta$, since $y = X\beta$. Therefore, the data generating process is summarized by the distribution of $X$, $Z$ and $\epsilon$ and by the vector $\beta$.
  2. Defining inputs for the model: this requires defining a multivariate distribution for $X$, $Z$ and $\epsilon$, the parameters for this distribution (means, variances and covariances), and defining values for $\beta$.
  3. Simulating outputs: values for $X$, $Z$ and $\epsilon$ should be drawn, allowing the generation of $y$, and, consequently, the calculation of $\hat{\beta}$. Since the probability convergence of $\hat{\beta}$ towards $\beta$ should be assessed, this simulation should take place over different values of $N$, the sample size.
  4. Analysis of results: finally, values of $\hat{\beta}$ are plotted against sample sizes, illustrating the probability convergence of IV estimates.

Theoretical references:
Wooldridge, J. Econometric Analysis of Cross Section and Panel Data, 2010.
Greene, W. Econometric Analysis, 2011.

In this notebook notebook we find libraries for Python implementation of the Monte Carlo experiment together with functions developed for executing the test. These functions provide the creation of a covariance matrix, given its parameters, the definition of the response variable $y$, given inputs $X$, parameters $\beta$, and random term $\epsilon$; the estimation of $\beta$ under IV, given by the equation above; the simulation of a data generating process, followed by an OLS estimation; and, finally, a function that fully implements the experiment. The remaining two sections run the experiment and assess the convergence by plotting estimates against sample sizes.

In addition to the consistency of IV estimates, the failure of OLS estimation to converge is also illustrated. Two additional experiments demonstrate how IV estimates are not consistent when instruments are not relevant (do not correlate with endogenous variables) and when instruments are not exogenous (correlate with the random error term).

def running_time(start_time, end_time, print_time=True):
    """
    Function that returns the amount of time for running a block of code.
    
    :param start_time: time point when the code was initialized.
    :type start_time: datetime object obtained by executing "datetime.now()".

    :param end_time: time point when the code stopped its execution.
    :type end_time: datetime object obtained by executing "datetime.now()".

    :param print_unit: unit of time for presenting the runnning time.
    :type print_unit: string.
    
    :return: prints start, end time and running times, besides of returning running time in seconds.
    :rtype: integer.
    """
    if print_time:
        print('------------------------------------')
        print('\033[1mRunning time:\033[0m ' + str(round(((end_time - start_time).total_seconds())/60, 2)) +
              ' minutes.')
        print('Start time: ' + start_time.strftime('%Y-%m-%d') + ', ' + start_time.strftime('%H:%M:%S'))
        print('End time: ' + end_time.strftime('%Y-%m-%d') + ', ' + end_time.strftime('%H:%M:%S'))
        print('------------------------------------')
    
    return (end_time - start_time).total_seconds()

def create_cov_matrix(variances, covariances):
    """
    Function that creates a covariance matrix from lists of variances and pair-wise covariances.
    
    :param variances: list with the variance of each input variable.
    :type variances: list or array.
    
    :param covariances: list with the pair-wise covariance between each combination of two input variables.
    :type covariances: list or array.
    
    :return: symmetric matrix whose elements in principal diagonal are variances and remaining elements are
    pair-wise covariances.
    
    :rtype: nd-array.
    """
    matrix = [[0 for j in range(len(variances))] for i in range(len(variances))]
    
    # Defining indices of the upper triangular part of the matrix:
    triu = np.triu_indices(4)
    indexes = [(i,j) for i, j in zip(triu[0], triu[1]) if i!=j]

    # Loop over indexes of the upper triangular part of the matrix:
    for idx in range(len(indexes)):
        i = indexes[idx][0]
        j = indexes[idx][1]

        # Defining the correspondent covariance:
        matrix[i][j] = covariances[idx]

    # Loop over indexes of the principal diagonal:
    for i in range(len(variances)):
        # Defining the correspondent variance:
        matrix[i][i] = variances[i]
        
    # Forcing the matrix to be symmetric:
    matrix = np.array(matrix)
    i_lower = np.tril_indices(len(variances), -1)
    matrix[i_lower] = matrix.T[i_lower]
    
    return matrix

def define_outcome(X, betas):
    """
    Function that generates values for the outcome variable given parameters of the data generating process.
    
    :param X: matrix with values for input variables (rows refer to observations, columns to variables).
    :type X: nd-array.
    
    :param betas: list with assumed populational coefficients of the linear regression model. Last element should
    be the intercept.
    :type betas: list or array.
    
    :return: list with generated values for the outcome variable.
    :rtype: array.
    """
    return np.dot(X, betas)

def estimate_betas_ols(X, y):
    """
    Function that estimates a vector of Ordinary Least Squares (OLS) estimates of coefficients of a populational
    linear regression model. It implements the following expression: betas_hat = (X^TX)^-1X^Ty.
    
    :param X: matrix with values for input variables (rows refer to observations, columns to variables).
    :type X: nd-array.
    
    :param y: list with values for the outcome variable.
    :type y: list.
    
    :return: list with estimates for the coefficients of the linear regression model.
    :rtype: array.
    """
    Xt = X.transpose()
    XtX = np.dot(Xt, X)
    XtX_inv = inv(XtX)
    XtX_invXt = np.dot(XtX_inv, Xt)

    return np.dot(XtX_invXt, y)

def estimate_betas_iv(X, Z, y):
    """
    Function that estimates a vector of Instrumental Variables (IV) estimates of coefficients of a populational
    linear regression model. It implements the following expression: betas_hat = (Z^TX)^-1Z^Ty.
    
    :param X: matrix with values for input variables (rows refer to observations, columns to variables). Within
    these input variables, it may be present some endogenous variables that should be instrumentalized.
    :type X: nd-array.
    
    :param Z: matrix with values for exogenous variables (rows refer to observations, columns to variables). It
    has the same dimension of X, but replaces the endogenous variables for the instruments, preserving the same
    number of columns.
    :type Z: nd-array.

    :param y: list with values for the outcome variable.
    :type y: list.
    
    :return: list with estimates for the coefficients of the linear regression model.
    :rtype: array.
    """
    Zt = Z.transpose()
    ZtX = np.dot(Zt, X)
    ZtX_inv = inv(ZtX)
    ZtX_invZt = np.dot(ZtX_inv, Zt)

    return np.dot(ZtX_invZt, y)

def simulate_iv(N, means, variances, covariances, betas, endo=[0], num_instruments=1, instrumental=True):
    """
    Function that produces IV/OLS estimation for simulated data.
    
    :param N: sample size.
    :rtype: integer.
    
    :param means: list with assumed populational means for input variables.
    :type means: list.
    
    :param variances: list with the variance of each input variable.
    :type variances: list or array.
    
    :param covariances: list with the pair-wise covariance between each combination of two input variables.
    :type covariances: list or array.
    
    :param betas: list with assumed populational coefficients of the linear regression model. Last element should
    be the intercept.
    :type betas: list or array.

    :param endo: list with indexes referring to endogenous variables present in the matrix of inputs.
    :type endo: list or array.
    
    :param num_instruments: number of instruments to be used. Here, it should be equal to the number of endogenous
    variables.
    :type num_instruments: integer.
    
    :param instrumental: indicates whether to implement OLS or IV estimation.
    :type instrumental: boolean.
    
    :return: list with estimates for the coefficients of the simulated linear regression model.
    :rtype: array.
    """
    # Creating the covariance matrix of input variables:
    cov_matrix = create_cov_matrix(variances=variances, covariances=covariances)

    # Defining the distribution of input variables:
    X_dist = multivariate_normal(mean=means, cov=cov_matrix)

    # Producing a matrix with sample values for the input variables:
    data = X_dist.rvs(N)
    
    # Matrix with relevant variables and the error term:
    n_cols = np.shape(data)[1]
    Xeps = data[:, [True if i < n_cols-num_instr else False for i in range(n_cols)]]

    # Matrix with relevant variables:
    X = Xeps[:, [True if i < np.shape(Xeps)[1]-1 else False for i in range(np.shape(Xeps)[1])]]

    # Matrix with only exogenous variables:
    Z = X.copy()
    Z[:, endo] = np.reshape(data[:, sorted([-1*i for i in range(1,num_instr+1)], reverse=False)],
                            (N,num_instr))

    # Including a column for the intercept:
    Xeps = np.hstack((Xeps, np.ones((len(Xeps), 1))))
    X = np.hstack((X, np.ones((len(X), 1))))
    Z = np.hstack((Z, np.ones((len(Z), 1))))

    # Defining the outcome variable from input variables, coefficients of linear model and the error term:
    betas = betas.copy()
    betas.insert(-1, 1)
    y = define_outcome(X=Xeps, betas=betas)

    return estimate_betas_iv(X=X, Z=Z, y=y) if instrumental else estimate_betas_ols(X=X, y=y)

def simulate(N_list, means, variances, covariances, betas, endo, num_instruments, instrumental):
    """
    Function that implements the simulation by generating data based on parameters of a data generating process,
    by estimating coefficients using IV/OLS, and by iterating over different sample sizes and over a (large)
    number of times.
    
    :param N_list: list of sample sizes.
    :type N_list: list.

    :param means: list with assumed populational means for input variables.
    :type means: list.
    
    :param variances: list with the variance of each input variable.
    :type variances: list or array.
    
    :param covariances: list with the pair-wise covariance between each combination of two input variables.
    :type covariances: list or array.

    :param betas: list with assumed populational coefficients of the linear regression model. Last element should
    be the intercept.
    :type betas: list or array.
    
    :param endo: list with indexes referring to endogenous variables present in the matrix of inputs.
    :type endo: list or array.
    
    :param num_instruments: number of instruments to be used. Here, it should be equal to the number of endogenous
    variables.
    :type num_instruments: integer.
    
    :param instrumental: indicates whether to implement OLS or IV estimation.
    :type instrumental: boolean.
    
    :return: dictionary with IV/OLS estimates, where keys are sample sizes and values are arrays with estimated
    coefficients.
    :rtype: dictionary.
    """
    start_time = datetime.now()
    bar_grid = progressbar.ProgressBar(maxval=len(N_list), widgets=['\033[1mSimulation progress:\033[0m ',
                                                                    progressbar.Bar('-', '[', ']'), ' ',
                                                                    progressbar.Percentage()])
    bar_grid.start()
    
    # Dictionary for storing IV/OLS estimates:
    outcomes = {}
    
    # Loop over sample sizes:
    for n in range(len(N_list)):
        # Storing IV estimates:
        outcomes[n] = simulate_iv(N=N_list[n], means=means, variances=variances, covariances=covariances,
                                  betas=betas, endo=endo, num_instruments=num_instruments,
                                  instrumental=instrumental)
        
        bar_grid.update(n+1)
        time.sleep(0.1)

    end_time = datetime.now()

    # Overall runing time:
    iteration_time = running_time(start_time=start_time, end_time=end_time, print_time=True)
    
    return outcomes

def create_plot(N_list, betas_hat, betas, color='black'):
    """
    Function that creates a line plot using plotly framework.
    
    :param N_list: list of sample sizes.
    :type N_list: list.
    
    :param betas_hat: dictionary with OLS estimates for each sample size.
    :type betas_hat: dictionary.

    :param betas: list with assumed populational coefficients of the linear regression model. Last element should
    be the intercept.
    :type betas: list or array.
    
    :return: displays a plotly line graph whose x-axis is given by sample sizes and whose y-axis represents the
    value of a coefficient.
    """
    # Create figure:
    fig = make_subplots(specs=[[{'secondary_y': False}]])

    fig.add_trace(
        go.Scatter(x = N_list,
                   y = betas_hat,
                   name = '',
                   hovertemplate = 'beta_hat = %{y:.4f}<br>' + 'N = %{x}<br>',
                   marker_color = color),
        secondary_y = False,
    )

    # Changing layout:
    fig.update_layout(
        title_text=f'Probability convergence of {"IV" if instrumental else "OLS"} estimate of beta{b.replace("b", "")}',
        width=600, height=450
    )

    # Set labels:
    fig.update_xaxes(title_text='N', tickangle=45)
    fig.update_yaxes(title_text=f'{"IV" if instrumental else "OLS"} estimate', secondary_y = False) # range=[0,4]

    fig.add_shape(type="line",
        x0=0, y0=betas, x1=max(N_list), y1=betas,
        line=dict(
            color="black",
            width=1,
            dash="solid",
        ))

    fig.show()
