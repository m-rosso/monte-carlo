Monte Carlo experiments¶

Probability convergence of OLS estimates¶

This project applies Monte Carlo experiments for assessing theoretical results. Here, the probability convergence of Ordinary Least Squares (OLS) estimates, i.e., their consistency property is illustrated making use of Monte Carlo methods that generate simulated data generating processes.

Our first assumption (OLS1) here is the linearity in parameters. Then, the vector of inputs $x$ (1 x p) and the vector of parameters $\beta$ (p x 1) relates linearly: $x\beta = \beta_1x_1 + \beta_2x_2 + ... + \beta_px_p + \beta_0$. Note that $y$ and $X$ do not necessarily need to also be linearly related: just assume that $x_2 = x_1^2$, for instance. Consequently, the fist assumption implies:

\begin{equation} \displaystyle y = x\beta + \epsilon.1 = \beta_0 + \beta_1x_1 + \beta_2x_2 + ... + \beta_px_p + \epsilon \end{equation}

Where $\epsilon$ is an assumed additive random error term. Note that this equation is valid for any observation in the population of interest. Given a random sample of $N$ instances: \begin{equation} \displaystyle y = X\beta + \epsilon \end{equation} Where, now, $y$ is N x 1, X is N x p, and $\epsilon$ is N x 1.

Another assumption (OLS2) for using OLS to estimate the assumed equation below is that $rank(E(x^Tx)^{-1}) = p + 1$. This is more of a technical constraint for getting an estimate $\hat{\beta}$ following the derivation below: \begin{equation} \displaystyle y = x\beta + \epsilon \end{equation} \begin{equation} \displaystyle x^Ty = x^Tx\beta + x^T\epsilon \end{equation} \begin{equation} \displaystyle E(x^Ty) = E(x^Tx)\beta + E(x^T\epsilon) \end{equation} \begin{equation} \displaystyle E(x^Tx)^{-1}E(x^Ty) = \beta + E(x^Tx)^{-1}E(x^T\epsilon) \end{equation}

Therefore, we find the last assumption (OLS3) needed for properly defining $\beta$:

\begin{equation} \displaystyle E(x^T\epsilon) = 0 \end{equation}

So: \begin{equation} \displaystyle \beta = E(x^Tx)^{-1}E(x^Ty) \end{equation}

This equation identifies $\beta$ through the method of moments, allowing its estimation by replacing expected values by averages:

\begin{equation} \displaystyle \hat{\beta} = \Big((1/N)\sum_{i=1}^Nx_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nx_i^Ty_i = (X^TX)^{-1}X^Ty \end{equation}

Above, we have developed the method of moments to identify $\beta$ and to derive estimates for these coefficients. The derived expression for $\hat{\beta}$ is equivalent to the solution of the minimization problem that leads to Ordinary Least Squares (OLS) estimates.

If we first assume hypotheses OLS1-OLS3 and then use that expression for $\hat{\beta}$, we find the theoretical result under which $\hat{\beta}$ converges in probability to $\beta$, i.e., $\hat{\beta}$ is consistent for estimating $\beta$:

\begin{equation} \displaystyle \hat{\beta} = \Big((1/N)\sum_{i=1}^Nx_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nx_i^Ty_i \end{equation} Using OLS1: \begin{equation} \displaystyle \hat{\beta} = \Big((1/N)\sum_{i=1}^Nx_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nx_i^T(x_i\beta + \epsilon_i) \end{equation} Organizing this expression leads to: \begin{equation} \displaystyle \hat{\beta} = \Big((1/N)\sum_{i=1}^Nx_i^Tx_i\Big)^{-1}(1/N)\Big(\sum_{i=1}^Nx_i^Tx_i\beta\Big) + \Big((1/N)\sum_{i=1}^Nx_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nx_i^T\epsilon_i \end{equation} \begin{equation} \displaystyle \hat{\beta} = \beta + \Big((1/N)\sum_{i=1}^Nx_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nx_i^T\epsilon_i \end{equation}
Letting $N \rightarrow \infty$: \begin{equation} \displaystyle \lim_{N \rightarrow \infty} \hat{\beta} = \beta + \lim_{N \rightarrow \infty}\Big((1/N)\sum_{i=1}^Nx_i^Tx_i\Big)^{-1}(1/N)\sum_{i=1}^Nx_i^T\epsilon_i \end{equation}

Given the delta method and the law of large numbers, and assuming OLS2:

\begin{equation} \displaystyle \lim_{N \rightarrow \infty} \hat{\beta} = \beta + E(x^Tx)^{-1}E(x^T\epsilon) \end{equation} Finally, OLS3 implies that: \begin{equation} \displaystyle \lim_{N \rightarrow \infty} \hat{\beta} = \beta \end{equation}

This proof demonstrates how OLS produces consistent estimates for $\beta$, following the inverse sequence of steps of the method of moments. However, both approaches assume the same hypotheses and, then, hold the same theoretical results.

Here, we apply general Monte Carlo simulation procedures to illustrate this property.

  1. Modeling the process that should be simulated: the process is depicted by the function $\hat{\beta} = (X^TX)^{-1}X^Ty$, which is a function of $X$, $\epsilon$ and $\beta$, since $y = X\beta$. Therefore, the data generating process is summarized by the distribution of $X$ and $\epsilon$ and by the vector $\beta$.
  2. Defining inputs for the model: this requires defining a multivariate distribution for $X$ and $\epsilon$, the parameters for this distribution (means, variances and covariances), and defining values for $\beta$.
  3. Simulating outputs: values for $X$ and $\epsilon$ should be drawn, allowing the generation of $y$, and, consequently, the calculation of $\hat{\beta}$. Since the probability convergence of $\hat{\beta}$ towards $\beta$ should be assessed, this simulation should take place over different values of $N$, the sample size.
  4. Analysis of results: finally, values of $\hat{\beta}$ are plotted against sample sizes, illustrating the probability convergence of OLS estimates.

Theoretical references:
Wooldridge, J. Econometric Analysis of Cross Section and Panel Data, 2010.
Greene, W. Econometric Analysis, 2011.

In this notebook notebook we find libraries for Python implementation of the Monte Carlo experiment together with functions developed for executing the test. These functions provide the creation of a covariance matrix, given its parameters, the definition of the response variable $y$, given inputs $X$, parameters $\beta$, and random term $\epsilon$; the estimation of $\beta$ under OLS, given by the equation above; the simulation of a data generating process, followed by an OLS estimation; and, finally, a function that fully implements the experiment. The remaining two sections run the experiment and assess the convergence by plotting estimates against sample sizes.

def running_time(start_time, end_time, print_time=True):
    """
    Function that returns the amount of time for running a block of code.
    
    :param start_time: time point when the code was initialized.
    :type start_time: datetime object obtained by executing "datetime.now()".

    :param end_time: time point when the code stopped its execution.
    :type end_time: datetime object obtained by executing "datetime.now()".

    :param print_unit: unit of time for presenting the runnning time.
    :type print_unit: string.
    
    :return: prints start, end time and running times, besides of returning running time in seconds.
    :rtype: integer.
    """
    if print_time:
        print('------------------------------------')
        print('\033[1mRunning time:\033[0m ' + str(round(((end_time - start_time).total_seconds())/60, 2)) +
              ' minutes.')
        print('Start time: ' + start_time.strftime('%Y-%m-%d') + ', ' + start_time.strftime('%H:%M:%S'))
        print('End time: ' + end_time.strftime('%Y-%m-%d') + ', ' + end_time.strftime('%H:%M:%S'))
        print('------------------------------------')
    
    return (end_time - start_time).total_seconds()

def create_cov_matrix(variances, covariances):
    """
    Function that creates a covariance matrix from lists of variances and pair-wise covariances.
    
    :param variances: list with the variance of each input variable.
    :type variances: list or array.
    
    :param covariances: list with the pair-wise covariance between each combination of two input variables.
    :type covariances: list or array.
    
    :return: symmetric matrix whose elements in principal diagonal are variances and remaining elements are
    pair-wise covariances.
    
    :rtype: nd-array.
    """
    matrix = [[0 for j in range(len(variances))] for i in range(len(variances))]
    
    # Defining indices of the upper triangular part of the matrix:
    triu = np.triu_indices(4)
    indexes = [(i,j) for i, j in zip(triu[0], triu[1]) if i!=j]

    # Loop over indexes of the upper triangular part of the matrix:
    for idx in range(len(indexes)):
        i = indexes[idx][0]
        j = indexes[idx][1]

        # Defining the correspondent covariance:
        matrix[i][j] = covariances[idx]

    # Loop over indexes of the principal diagonal:
    for i in range(len(variances)):
        # Defining the correspondent variance:
        matrix[i][i] = variances[i]
        
    # Forcing the matrix to be symmetric:
    matrix = np.array(matrix)
    i_lower = np.tril_indices(len(variances), -1)
    matrix[i_lower] = matrix.T[i_lower]
    
    return matrix

def define_outcome(X, betas, eps):
    """
    Function that generates values for the outcome variable given parameters of the data generating process.
    
    :param X: matrix with values for input variables (rows refer to observations, columns to variables).
    :type X: nd-array.
    
    :param betas: list with assumed populational coefficients of the linear regression model. Last element should
    be the intercept.
    :type betas: list or array.
    
    :param eps: list with randomly generated error terms.
    :type eps: list or array.
    
    :return: list with generated values for the outcome variable.
    :rtype: array.
    """
    return np.dot(X, betas) + eps

def estimate_betas(X, y):
    """
    Function that estimates a vector of Ordinary Least Squares (OLS) estimates of coefficients of a populational
    linear regression model. It implements the following expression: betas_hat = (X^TX)^-1X^Ty.
    
    :param X: matrix with values for input variables (rows refer to observations, columns to variables).
    :type X: nd-array.
    
    :param y: list with values for the outcome variable.
    :type y: list.
    
    :return: list with estimates for the coefficients of the linear regression model.
    :rtype: array.
    """
    Xt = X.transpose()
    XtX = np.dot(Xt, X)
    XtX_inv = inv(XtX)
    XtX_invXt = np.dot(XtX_inv, Xt)

    return np.dot(XtX_invXt, y)

def simulate_ols(N, means, variances, covariances, betas):
    """
    Function that produces OLS estimation for simulated data.
    
    :param N: sample size.
    :rtype: integer.
    
    :param means: list with assumed populational means for input variables.
    :type means: list.
    
    :param variances: list with the variance of each input variable.
    :type variances: list or array.
    
    :param covariances: list with the pair-wise covariance between each combination of two input variables.
    :type covariances: list or array.
    
    :param betas: list with assumed populational coefficients of the linear regression model. Last element should
    be the intercept.
    :type betas: list or array.
    
    :return: list with estimates for the coefficients of the simulated linear regression model.
    :rtype: array.
    """
    # Creating the covariance matrix of input variables:
    cov_matrix = create_cov_matrix(variances=variances, covariances=covariances)

    # Defining the distribution of input variables:
    X_dist = multivariate_normal(mean=means, cov=cov_matrix)

    # Defining the distribution of the (additive) random error term:
    error_dist = norm(loc=0, scale=1)

    # Producing a matrix with sample values for the input variables:
    X = X_dist.rvs(N)

    # Including a column for the intercept:
    X = np.hstack((X, np.ones((len(X), 1))))

    # Producing a list of sample values for the (additive) random error term:
    eps = error_dist.rvs(N)

    # Defining the outcome variable from input variables, coefficients of linear model and the error term:
    y = define_outcome(X=X, betas=betas, eps=eps)

    return estimate_betas(X=X, y=y)

def simulate(N_list, means, variances, covariances, betas):
    """
    Function that implements the simulation by generating data based on parameters of a data generating process,
    by estimating coefficients using OLS, and by iterating over different sample sizes.
    
    :param N_list: list of sample sizes.
    :type N_list: list.

    :param means: list with assumed populational means for input variables.
    :type means: list.
    
    :param variances: list with the variance of each input variable.
    :type variances: list or array.
    
    :param covariances: list with the pair-wise covariance between each combination of two input variables.
    :type covariances: list or array.

    :param betas: list with assumed populational coefficients of the linear regression model. Last element should
    be the intercept.
    :type betas: list or array.
    
    :return: dictionary with OLS estimates, where keys are sample sizes and values are arrays with estimated
    coefficients.
    :rtype: dictionary.
    """
    start_time = datetime.now()
    bar_grid = progressbar.ProgressBar(maxval=len(N_list), widgets=['\033[1mSimulation progress:\033[0m ',
                                                                    progressbar.Bar('-', '[', ']'), ' ',
                                                                    progressbar.Percentage()])
    bar_grid.start()
    
    # Dictionary for storing OLS estimates:
    outcomes = {}

    # Loop over sample sizes:
    for n in range(len(N_list)):
        # Storing OLS estimates:
        outcomes[n] = simulate_ols(N=N_list[n], means=means, variances=variances, covariances=covariances,
                                   betas=betas)
        
        bar_grid.update(n+1)
        time.sleep(0.1)

    end_time = datetime.now()

    # Overall runing time:
    iteration_time = running_time(start_time=start_time, end_time=end_time, print_time=True)
    
    return outcomes
